Aiffd2007

R versions of boxed examples for the Analysis and Interpretation of Freshwater Fisheries Data (2007)

Download this project as a .zip file Download this project as a tar.gz file

The following are R versions of the “boxed example analyses” in the Analysis and Interpretation of Freshwater Fisheries Data book (Guy and Brown, 2007). Not all chapters, nor all boxes within each chapter, have been completed.

The documents were recently converted to RMarkdown and some code was updated. It is unlikely that the conversion was perfect, so let me know if you find omissions or errors.
