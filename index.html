Aiffd2007

R versions of boxed examples for the Analysis and Interpretation of Freshwater Fisheries Data (2007)

Download this project as a .zip file Download this project as a tar.gz file

The following are R versions of the “boxed example analyses” in the Analysis and Interpretation of Freshwater Fisheries Data book (Guy and Brown, 2007).

Not all chapters have been completed and not all boxes within each chapter below have been completed. We recently converted the documents to RMarkdown and updated some code that was out-of-date. The conversion was unlikely to be perfect so let me know if you find omissions or errors.
